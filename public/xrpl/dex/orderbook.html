XRPL Order Book Viewer

Look Up Order Book for a Token Pair

Use this page to inspect the sell and buy offers for a given currency pair on the XRP Ledger’s built‑in DEX. By default, this uses the XRPL Testnet.

XRPL Network, Token Pair and Ledger

Defaults to XRPL Testnet. You can change this to another XRPL network URL if needed.
Token 1 is the base asset in the pair. If you enter XRP, leave issuer empty; otherwise provide an issuing address.
Token 2 is the counter asset in the pair.
Use validated (default) to query the latest validated ledger, or enter a specific ledger index or shortcut supported by the server.

Sell and Buy Offers

Sell offers are offers where traders are selling Token 1 for Token 2. Buy offers are the opposite direction (selling Token 2 for Token 1).
